4. What is DNS and when does it come into the picture?

Domain Name System (DNS) translates human-readable domain names (like www.example.com) into IP addresses (like 192.0.2.1), which computers use to identify each other on the network. It comes into play when you type a domain name into your browser's address bar.

5. What is markup?

Markup is a set of codes that define a document's structure and presentation, like HTML for web pages.

6. SEO (Knowledge of basics)

Search Engine Optimization (SEO) in HTML involves using HTML tags to improve a website's visibility in search engines.

7. Inspect Element

"Inspect Element" is a browser tool that allows you to view and modify the HTML, CSS, and JavaScript of a webpage in real-time.

8. Explain what webpage designing is.

Web page design in HTML refers to the process of creating the structure and layout of a webpage using the HyperText Markup Language (HTML), which defines the basic elements like headings, paragraphs, images, links, and more.

9. How will you open another window using an anchor tag?

To open a webpage in a new window (or tab) using an anchor (<a>) tag in HTML, use the target="_blank" attribute.

10. What does the viewport meta tag do?

The viewport meta tag controls the layout and scaling of a webpage on different devices, ensuring it is properly responsive and adapts to various screen sizes. Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">

11. What are Attributes in HTML?

Attributes in HTML provide additional information about HTML elements, defining properties like behavior, appearance, and functionality. They are placed within the opening tag of an element.

12. What is DNS?

DNS (Domain Name System) is a system that translates domain names into IP addresses, enabling browsers to load websites.

13. What is an IP address?

An IP address (Internet Protocol address) is a unique number that identifies a device connected to the internet.

14. What is localhost?

Localhost is a hostname that refers to the local computer or device, used to access network services running on the same machine.

15. HTML5 vs HTML4

HTML5 is the latest version of HTML, offering improved multimedia support, semantic elements, and better performance, while HTML4 is the older standard with limited features for modern web development.

16. What are semantic tags?

Semantic tags in HTML are elements that clearly describe their meaning in the context of the content, such as <header>, <footer>, <nav>, etc.

17. What are the new features in HTML5?

Some new features in HTML5 include:

  1. Semantic Elements: New tags like <article>, <section>, and <header> for better page structure.
  2. Multimedia Support: Native <audio> and <video> tags for embedding media.
  3. Canvas Element: For drawing graphics and animations with JavaScript.
  4. Local Storage & Session Storage: To store data on the user's device.

18. Difference between div and span

The <div> tag is a block-level element used to group content and create sections, while the <span> tag is an inline element used to style or group small portions of text or other inline elements.

19. Difference between class and IDs

id is a unique identifier for a single element, while class is used to group multiple elements together for shared styling or behavior.

20. What are self-closing tags?

Self-closing tags are HTML tags that don't require a closing tag because they contain no content, such as <img>, <input>, and <br>.

21. How many types of headings are there? How to decide which to use where?

There are 6 types of headings (<h1> to <h6>), with <h1> for the main title and subsequent tags for subheadings in descending order of importance.

22. Difference between block and inline

Block elements occupy the entire width and stack vertically (e.g., <div>, <h1>), while inline elements only take up the space they need and stay within the flow of text (e.g., <span>, <strong>).

23. Difference between em and strong

<em> is used to emphasize text (usually rendered in italics), while <strong> is used to indicate strong importance (usually rendered in bold).

24. What are attributes?

Attributes are additional information provided within an HTML tag that modify its behavior or provide extra details, such as src for an image source or href for a hyperlink.

25. Difference between HTML tags and elements

An HTML tag is the markup (e.g., <div>), while an HTML element includes the tag, content, and attributes (e.g., <div class="box">Content</div>).

26. Explain br tags

The <br> tag in HTML is a self-closing tag used to insert a line break, creating a new line within text or content without starting a new paragraph.

27. What are non-semantic tags?

A non-semantic tag in HTML is an element that does not convey meaning about its content, such as <div> and <span>, which are used for layout and styling but don't describe the type of content inside.
