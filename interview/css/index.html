1. What is Box Sizing?

The box-sizing property defines how the width and height of an element are calculated: should they include padding and borders, or not.

2. What is Animation Delay?

The animation-delay property specifies a delay for the start of an animation.The animation-delay value is defined in seconds (s) or milliseconds (ms).

3. Which property you will use to merge cells vertically in grid?

To merge cells vertically in a CSS Grid layout, you can use the grid-row property. Specifically, you set the grid-row to span across multiple rows.

4. Which property you will use to merge cells horizontally in grid?

To merge cells horizontally in a CSS Grid layout, you use the grid-column property. By setting this property, you can make a cell span across multiple columns

5. Explain the CSS Box Model

The CSS Box Model is a fundamental concept in web design and layout. It describes the rectangular boxes generated for elements in the document, including their dimensions and how they interact with other elements on the page.

The CSS Box Model consists of:

  • Content:The actual content (e.g., text or images).
  • Padding: Space between the content and the border.
  • Border: Surrounds the padding and content.
  • Margin: Space outside the border, separating the element from others.

6. What is the difference between display: none and visibility: hidden?

  • display: none: The element is completely removed from the layout and doesn't take up any space.
  • visibility: hidden The element is hidden but still takes up space in the layout.
