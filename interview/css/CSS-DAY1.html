CSS Interview Questions

1. Difference Between Flex Row and Column

In a Flexbox layout, the flex-direction property determines whether the flex container's children are laid out in a row or column.

  • Flex Row: row arranges items horizontally (left to right).
  • Flex Column: column arranges items vertically (top to bottom).

2. Explain Inline, Internal, and External Stylesheets

There are three ways to apply CSS to an HTML document:

  • Inline CSS: Written directly in the HTML element using the style attribute.
  • Internal CSS: Defined within the <style> tag in the <head> section of the HTML document.
  • External CSS: Defined in an external file (usually with a .css extension) and linked to the HTML document using the <link> tag.

3. Justify-Content allow you to do what?

The justify-content property is used to align the items in a flex container along the main axis (horizontally in a row or vertically in a column).

  • justify-content: flex-start; Aligns items to the start of the flex container.
  • justify-content: center; Aligns items to the center of the flex container.
  • justify-content: space-between; Distributes items evenly with space between.
  • justify-content: space-around; Distributes items evenly with space around them.

4. Difference Between Absolute and Relative Positioning

The position property determines how an element is positioned in relation to its parent container or the viewport:

  • Absolute Positioning: The element is positioned relative to the nearest positioned ancestor (not static), or the initial containing block (usually the viewport).
  • Relative Positioning: The element is positioned relative to its normal position in the document flow, and it can be adjusted using top, left, bottom, and right.

5. What is Z-Index in CSS?

The z-index property in CSS determines the stacking order of elements. Elements with a higher z-index value will appear in front of elements with a lower z-index.

Only elements that have a position other than static (such as relative, absolute, or fixed) can have a z-index.

6. Difference Between Padding and Margin

Padding is the space between the content of an element and its border, whereas margin is the space between the element's border and surrounding elements.

  • Padding: Adds space inside the element's border.
  • Margin: Adds space outside the element's border.
